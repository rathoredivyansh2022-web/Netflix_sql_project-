# Netflix Movies & TV Shows Data Analysis

![Netflix logo](https://raw.githubusercontent.com/rathoredivyansh2022-web/Netflix_sql_project-/main/logo.png)

## Overview
This project analyzes Netflix's movies and TV shows data using SQL to extract insights and answer business questions about content distribution, ratings, and trends.

## Objectives
- Analyze the distribution of Movies vs TV Shows
- Identify the most common ratings for different content types
- Examine content trends by year, country, and duration
- Categorize and classify content based on various attributes

## Dataset
*Source*: [Kaggle - Netflix Movies and TV Shows Dataset](https://www.kaggle.com/)

The dataset includes information about show titles, directors, cast, countries, release dates, ratings, duration, genres, and descriptions.

## Analysis Performed

1. *Content Type Analysis* - Compared the total count of Movies vs TV Shows
2. *Rating Distribution* - Found the most common ratings for each content type
3. *Release Year Filtering* - Listed all movies released in specific years
4. *Geographic Analysis* - Identified top countries producing Netflix content
5. *Duration Analysis* - Found the longest movies and TV shows with most seasons
6. *Temporal Analysis* - Analyzed content added in recent years
7. *Director & Cast Analysis* - Searched for content by specific directors and actors
8. *Genre Classification* - Counted content across different genres
9. *Country-Specific Insights* - Analyzed content release trends for specific countries
10. *Content Categorization* - Classified content based on keywords and themes

## Key Findings

- Netflix maintains a diverse library with both movies and TV shows
- Certain ratings are more common for specific content types
- India is a major content contributor to the platform
- Content production has increased significantly in recent years
- Genre distribution shows varied content to cater to different audiences

## Technologies Used
- *Database*: PostgreSQL
- *Query Language*: SQL
- *Tools*: Data analysis using advanced SQL queries including CTEs, window functions, and string operations
